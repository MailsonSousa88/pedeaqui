# US-LJ-0001 - Cadastrar e configurar loja

## Épico

Épico 2 - Gestão de Loja pelo Lojista.

## Ator

Lojista.

## História de usuário

Como lojista, eu quero cadastrar e configurar as informações da minha loja para que eu possa disponibilizar meu negócio na plataforma.

## Valor para o usuário

Permite que o lojista crie sua vitrine digital e apresente as informações essenciais do negócio para os clientes.

## Critérios de aceitação

- Dado que o lojista esteja autenticado, quando acessar a configuração da loja, então o sistema deve permitir o preenchimento das informações da loja.
- Dado que o lojista preencha os dados obrigatórios, quando salvar as informações, então o sistema deve cadastrar ou atualizar a loja.
- Dado que existam campos obrigatórios ausentes, quando o lojista tentar salvar, então o sistema deve impedir o salvamento e informar os campos pendentes.
- Dado que as informações sejam atualizadas, quando a vitrine pública for acessada, então o sistema deve exibir os dados atualizados.

## Prioridade

Alta.

## Dependências

- Cadastro válido do lojista.
- Acesso ao painel da loja.
