Este documento define os padrões de organização e documentação que devem ser seguidos neste repositório pelos colaboradores

## 📁 Estrutura de documentação (Frontend)

>Toda documentação do projeto deve ser adicionada na pasta **docs/**

A estrutura inicial deve seguir esse padrão:

```plaintext
docs/
  requisitos/
    funcionais/
    nao-funcionais/
```

⚠️ Observação: Essa estrutura vai mudar a medida que novos artefatos estiverem concluídos
