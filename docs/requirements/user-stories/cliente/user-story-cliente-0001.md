# US-CL-0001 - Visualizar produtos de uma loja

## Épico

Épico 1 - Jornada de Compra do Consumidor.

## Ator

Cliente.

## História de usuário

Como cliente, eu quero acessar uma loja e visualizar seus produtos para que eu possa conhecer o que o lojista oferece.

## Valor para o usuário

Permite que o cliente consulte a vitrine digital de uma loja antes de iniciar contato direto com o lojista.

## Critérios de aceitação

- Dado que existem lojas ativas na plataforma, quando o cliente acessar uma loja, então o sistema deve exibir as informações públicas da loja.
- Dado que a loja possui produtos cadastrados, quando o cliente acessar a vitrine, então o sistema deve listar os produtos disponíveis.
- Dado que um produto esteja indisponível ou removido, quando a vitrine for carregada, então esse produto não deve aparecer como disponível para pedido.

## Prioridade

Alta.

## Dependências

- Loja cadastrada e ativa.
- Produtos cadastrados pelo lojista.
