# US-CL-0002 - Buscar lojas na plataforma

## Épico

Épico 1 - Jornada de Compra do Consumidor.

## Ator

Cliente.

## História de usuário

Como cliente, eu quero navegar e buscar por diferentes lojas dentro da plataforma para que eu possa encontrar lojas do meu interesse.

## Valor para o usuário

Facilita a descoberta de lojas disponíveis e melhora a experiencia de navegação do cliente.

## Critérios de aceitação

- Dado que existem lojas ativas, quando o cliente acessar a página inicial, então o sistema deve exibir a listagem de lojas disponíveis.
- Dado que o cliente utilize busca ou filtro, quando houver lojas correspondentes, então o sistema deve exibir os resultados encontrados.
- Dado que não existam lojas correspondentes, quando o cliente realizar uma busca, então o sistema deve informar que nenhum resultado foi encontrado.

## Prioridade

Alta.

## Dependências

- Lojas cadastradas e ativas.
- Informações públicas das lojas disponíveis para consulta.
