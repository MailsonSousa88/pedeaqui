# Spacing

## Escala

Usar escala baseada em 4px: `1`, `2`, `3`, `4`, `6`, `8`, `10`, `12`, `16`.

## Regras

- Formulários devem usar gaps consistentes entre campos.
- Cards de formulário devem ter padding confortável em mobile e desktop.
- Evitar layouts com espaçamento decorativo excessivo em telas operacionais.
