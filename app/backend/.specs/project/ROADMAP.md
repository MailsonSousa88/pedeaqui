# Project Roadmap

## Milestone 1: Authentication & User Profiles (Current)
Implement secure authentication flow and profile data management using Supabase Auth and PostgreSQL.

### Features
- [ ] Sign Up (Email and Password)
- [ ] Login (Email and Password)
- [ ] Logout
- [ ] Password Recovery
- [ ] Persistent Session (JWT validation middleware)
- [ ] Protected Routes
- [ ] Profiles table for additional user data

## Milestone 2: Tenant & Store Management
Allow authenticated users to create and configure their tenant profile and digital storefront.

## Milestone 3: Catalog & Products
Implement categories, products, images, and variation options.

## Milestone 4: Checkout & Order Redirection
Implement WhatsApp order redirection and final checkout details.
