# US-AD-0001 - Gerenciar usuários lojistas

## Épico

Épico 3 - Administração e Monitoramento da Plataforma.

## Ator

Administrador.

## História de usuário

Como administrador, eu quero gerenciar os usuários lojistas, aprovando, bloqueando e editando contas, para que eu possa garantir que apenas lojistas válidos utilizem a plataforma.

## Valor para o usuário

Permite que a administração mantenha controle sobre os lojistas cadastrados e reduza riscos operacionais, legais e de qualidade.

## Critérios de aceitação

- Dado que existam lojistas cadastrados, quando o administrador acessar a aba de lojistas, então o sistema deve exibir a listagem de contas.
- Dado que o administrador utilize busca ou filtros, quando houver resultados, então o sistema deve exibir os lojistas correspondentes.
- Dado que o administrador selecione um lojista, quando abrir os detalhes, então o sistema deve exibir as informações da conta.
- Dado que o administrador aprove, bloqueie ou edite uma conta, quando confirmar a operação, então o sistema deve atualizar a conta do lojista.
- Dado que uma ação administrativa seja realizada, quando a operação for concluída, então o sistema deve registrar a ação.

## Prioridade

Alta.

## Dependências

- Painel administrativo.
- Cadastro de lojistas.
- Registro de ações administrativas.
