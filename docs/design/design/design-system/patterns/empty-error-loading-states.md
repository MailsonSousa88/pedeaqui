# Empty, Error and Loading States

## Loading

- Mostrar estado visível sem deslocar layout de forma brusca.
- Bloquear ação duplicada durante submissão.

## Empty

- Explicar o estado vazio e oferecer próxima ação quando existir.

## Error

- Mensagem em português, clara e acionável.
- Erros de campo perto do campo.
- Erros gerais dentro do container da tela.
